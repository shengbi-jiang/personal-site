  # Factory Method
  Let's say we want to create web content with a title and a URL. The page should use the URL to fetch the source data for its content. The content could be some text, an image, or a video. It should be easy for developers to create new types of content and swap them. We can use Factory Method pattern to achieve it. The class `Content` defines the outline of the content, and its child classes implement types of content. For example, `ArticleContent` fetches an article content with the URL, and `ImageContent` uses its image component and the URL to show the picture.
